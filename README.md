# CUDA by Example Practice

This repository contains my practice code while working through the book **CUDA by Example**.

## Structure

- `chapter-03/` → examples for Chapter 3
- `chapter-04/` → examples for Chapter 4
- each `example-*` folder contains a CUDA program for that example

## How to Build

Go to an example folder and run:

```bash
nvcc main.cu -o main
./main
```
